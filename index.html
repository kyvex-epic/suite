Kyvix Suite

Kyvex Suite is a FOSS alternative to big tech. We provide a suite of services that are privacy respecting, open source, and free. We are a small team of developers that are passionate about privacy and open source software. We believe that everyone should have access to the internet without being tracked and sold to. We also believe that everyone should have access to the source code of the software they use. We are here to provide that.

Features

Kyvex Cloud

Kyvex Cloud is a cloud storage service that is privacy respecting, open source, and free. We provide a gig of free storage to all users.

Kyvex Docs

Kyvex Docs is a document editor that is privacy respecting, open source, and free. We provide a full suite of document editing tools. We also provide a paid plan for users that need more storage.

Kyvex Code

Kyvex Code is a code editor that is privacy respecting, open source, and free. We provide a full suite of code editing tools. We also provide a paid plan for users that need more storage.
